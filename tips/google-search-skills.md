# Google Search Skills

Google search skills from this YouTube [video](https://www.youtube.com/watch?v=7ond5eF7L-I).

### `related`

```text
related:google.com
```

### `filetype`

```text
sicp filetype:pdf
```

### search operators: `+`, `-`

```text
tom cruise +scientology

tom cruise -scientology
```

### `*`

```text
may the * be with you
```

### `nearby`

```text
coffee nearby
```
